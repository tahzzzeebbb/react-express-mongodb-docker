# ⚛️ React + Express + MongoDB Docker

A full-stack MERN application containerized with Docker and Docker Compose. This project demonstrates multi-container deployment by integrating a React frontend, Express.js backend, and MongoDB database into a single Dockerized environment.

---

## 📖 Project Overview

This project was developed as part of a Cloud Computing coursework assignment to demonstrate containerization of a MERN stack application using Docker. Docker Compose orchestrates communication between the frontend, backend, and MongoDB database.

---

## ✨ Features

- ⚛️ React Frontend
- 🚀 Express.js REST API
- 🍃 MongoDB Database
- 🐳 Docker Containerization
- 🔄 Docker Compose
- 🌐 Multi-container Architecture
- 📦 Persistent Database Storage

## 🚀 Getting Started

### Build Containers

```bash
docker-compose build
```

### Start Containers

```bash
docker-compose up
```
